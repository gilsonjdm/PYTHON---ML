# 🐍 Aula 01 — Introdução ao Python

## 📌 O que é Python

Python é uma **linguagem de programação de alto nível**, **interpretada**, **multiplataforma** e de **propósito geral**.

Ela foi criada com foco em:

* simplicidade
* legibilidade do código
* produtividade do desenvolvedor

Por isso é considerada uma das melhores linguagens para **iniciantes em programação**.

---

## 🧠 Principais usos do Python

Python é utilizada em várias áreas da computação:

* 🌐 Desenvolvimento web
* ⚙️ Automação de tarefas
* 📊 Análise e ciência de dados
* 🤖 Inteligência artificial e Machine Learning
* 🔬 Computação científica
* 🎓 Ensino de programação

---

## 👨‍💻 História do Python

Python foi criado no final dos anos **1980** por **Guido van Rossum** no instituto **CWI (Centrum Wiskunde & Informatica)** na Holanda.

Curiosidades:

* O nome **Python** vem do grupo de comédia britânico **Monty Python**.
* A primeira versão oficial (**Python 0.9.0**) foi lançada em **1991**.
* Em **2008** foi lançado o **Python 3.0**, que se tornou o padrão atual da linguagem.

## ⭐ Por que usar Python

### 1️⃣ Simplicidade

A sintaxe do Python é clara e próxima da linguagem humana, facilitando a leitura e aprendizado.

### 2️⃣ Alta produtividade

Problemas complexos podem ser resolvidos com **poucas linhas de código**.

### 3️⃣ Grande comunidade

Python possui:

* muitas bibliotecas
* vasta documentação
* grande quantidade de cursos e materiais

### 4️⃣ Uso na indústria

Empresas que utilizam Python incluem:

* Google
* Netflix
* Spotify
* Amazon
* Meta

Principalmente em **dados e inteligência artificial**.

### 5️⃣ Uso acadêmico

Python é muito utilizado em:

* universidades
* pesquisa científica
* ensino de programação
* ciência de dados

## 🧰 Principais áreas de aplicação

Algumas áreas onde Python se destaca:

* **Automação:** scripts para tarefas repetitivas
* **Web:** frameworks como Flask e Django
* **Ciência de dados:** análise e visualização de dados
* **Machine Learning:** treinamento de modelos
* **Inteligência Artificial:** visão computacional e NLP

---

## 💻 Como usar Python

### Instalação

O Python pode ser baixado no site oficial:

```
https://www.python.org
```

Disponível para:

* Windows
* Linux
* macOS

---

### Executando código Python

Existem duas formas principais:

1. **Modo interativo**

   * executa comandos linha por linha

2. **Arquivos `.py`**

   * scripts completos

Exemplo de código:

```python
print("Olá, mundo!")
```
